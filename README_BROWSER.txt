Kali Browser - Installation and Usage Instructions

REQUIREMENTS:
- Python 3.8 or higher
- pip package manager

INSTALLATION:
1. Install Python dependencies:
   pip install -r requirements.txt

RUNNING THE BROWSER:
   python kali_browser.py

FEATURES:
- Modern, secure web browsing with QtWebEngine
- Navigation controls (back, forward, reload, home)
- Address bar with URL validation
- HTTPS enforcement for secure connections
- Sandboxed iframe protection
- Multiple tab support
- Loading animations and smooth transitions
- Support for all standard video/image formats via native rendering

SECURITY:
- Automatic HTTPS upgrade warnings
- URL validation and sanitization
- Sandboxed web content
- Built-in web engine security protections
